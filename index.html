SlashRoles

SlashRoles is a slash command bot that lets you setup options in the Slash Command menu to assign roles. By running /create, you can select a role to assign, and the title and description to show in the Slash Command menu. When someone runs that command, they'll be granted the role you selected. You can remove an option by using /delete, and the data related to it will be deleted.
